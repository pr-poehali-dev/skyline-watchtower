# Дискордик — Figma Rich Presence для Discord

![Preview](https://cdn.poehali.dev/templates/diskordik-ru/preview-ru.jpg)

Лендинг для приложения, которое показывает ваш статус работы в Figma прямо в Discord. Выполнен в стиле интерфейса Discord с боковыми панелями, каналами и демонстрацией Rich Presence. Отличный выбор для продвижения утилит и расширений для дизайнеров.
